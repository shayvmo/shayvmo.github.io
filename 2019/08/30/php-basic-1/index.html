PHP字符串定义方式及各自区别

【定义方式】
单引号
双引号

newdoc :

【区别】

单引号不解析变量，
单引号不解析转义字符，只能解析单引号和反斜线本身

双引号解析变量、所有转义符号
变量可以使用特殊字符和{}包含

同等情况下，单引号效率更高

heredoc 相当于多行双引号

nowdoc 类似于多行单引号

【数据类型】

浮点型

不能运用到比较运算中，会有单双精度的问题。

布尔型

FALSE的多种情况

数组类型

超全局数组

NULL 类型

3种情况，NULL，未定义变量，unset后的变量

常量

const 语言结构
define 函数

2者相比，const 更快
