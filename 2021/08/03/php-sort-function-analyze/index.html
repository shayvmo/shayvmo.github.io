PHP 数组排序函数 sort 底层实现分析

PHP源码 github 仓库地址：https://github.com/php/php-src

源码路径：

ext\standard\array.c

Zend\zend_hash.h

Zend\zend_sort.c

从上面源码可以看出，sort 函数当排序的数组元素小于等于 16 个时，采用插入排序的方式进行排序；大于 16 个元素时，采用快速排序的方式。

当元素个数小于等于 5 个时，直接使用定义的函数进行简单排序交换。
